```
## 🔄 **TARE SCALE BUTTON (BMI Only):**
```
┌─────────────────────────────────┐
│ Weight                          │
│ 65.5 kg              [Scale]    │
│                                 │
│ ┌─────────────────────────────┐ │
│ │  [🔄] Tare Scale            │ │ ← THIS BUTTON
│ └─────────────────────────────┘ │
```

**When you press "Tare Scale":**
1. ESP32 runs `weightScale.tare(20)`
2. Takes 20 readings
3. Sets current weight as ZERO
4. Shows snackbar: "✅ Scale tared successfully!"

## 📊 **DATA FLOW:**
```
User clicks "Start Measurement"
         ↓
Shows "Connecting to Sensor..." (1.5 seconds)
         ↓
Starts polling ESP32 every 500ms
         ↓
Updates live data on screen
         ↓
Shows colored status indicators
         ↓
User clicks "Save Data"
         ↓
Calls onDataSaved(finalData)
         ↓
Shows "✅ BMI data saved!" snackbar
